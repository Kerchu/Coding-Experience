"A shooter fired 20 shots at this target. Each shot has a score of 100, 50, 10 and 0
points respectively according to whether you have hit the center, in the middle, at the end or you have failed
Write a program to calculate the shooter's total score and print it out. account for that
with a file, where in each component only one trigger value (D) is saved. If the file has more
of 20 items, print a warning message and continue to display the overall score."

Program oriented to stock the points made by players while shooting at a target
